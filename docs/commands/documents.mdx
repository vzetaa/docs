## Normal usage

### Hiragana List

Randomly list Hiragana characters.

```tsx
Command : !hiragana
Aliases : !hira
Usage   : !hiragana [randomNum: Number]
Example : !hiragana 34
```

### Katakana List

Randomly list Katakana characters.

```tsx
Command : !katakana
Aliases : // No aliases.
Usage   : !katakana [randomNum: Number]
Example : !katakana 34
```
